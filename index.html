1
संधि (Sandhi)
🔥 5–7 Qहर साल
›
💡 Concept
दो शब्दों के मेल पर होने वाला ध्वनि-परिवर्तन संधि कहलाता है। यह 3 प्रकार की होती है: स्वर, व्यंजन, विसर्ग। संधि में अक्षर बदलते हैं — विच्छेद करने पर मूल शब्द मिलते हैं।
स्वर संधि (अच् संधि) व्यंजन संधि (हल् संधि) विसर्ग संधि
📐 Rules
अ/आ + अ/आ = आ → देव+आलय = देवालय
अ/आ + इ/ई = ए → देव+इंद्र = देवेंद्र
अ/आ + उ/ऊ = ओ → सूर्य+उदय = सूर्योदय
इ/ई + कोई = य्+ → यदि+अपि = यद्यपि
उ/ऊ + कोई = व्+ → मनु+अंतर = मन्वंतर
अ + ए/ओ = ऐ/औ → लोक+एषणा = लोकैषणा
⚡ Trick
RULE याद करें: "आ-ए-ओ-ऐ-औ" = स्वर सीढ़ी
• जहाँ ण् या न् आए = व्यंजन संधि
• जहाँ : (विसर्ग) था = विसर्ग संधि
• विच्छेद करते समय शब्द तोड़कर लिखें, क्रम सही रखें
📝 Exam Q
"विद्यार्थी" में कौन सी संधि है?
(A) व्यंजन (B) विसर्ग (C) दीर्घ स्वर (D) गुण
✅ (C) विद्या+अर्थी → आ+अ = आ (दीर्घ)

समास (Samas — Compound Words)
🔥 4–5 Q
›
💡 Concept
दो या अधिक शब्दों का संक्षिप्त रूप समास है। विभक्ति लोप होती है। 6 प्रकार होते हैं।
तत्पुरुषकर्मधारयद्विगु द्वंद्वबहुव्रीहिअव्ययीभाव
📐 6 Types + Trick
तत्पुरुष: दूसरा पद प्रधान · "का/के/की" जोड़ें
कर्मधारय: विशेषण+विशेष्य · "जो है वही"
द्विगु: संख्या पूर्वपद · "त्रिभुज, पंचवटी"
द्वंद्व: दोनों पद प्रधान · "और/या" जोड़ें
बहुव्रीहि: दोनों अप्रधान · तीसरा अर्थ
अव्ययीभाव: पहला अव्यय · "यथाशक्ति"
⚡ TRICK — "तकदबाव"
त=तत्पुरुष, क=कर्मधारय, द=द्विगु,
ब=बहुव्रीहि, आ=अव्ययीभाव, व=द्वंद्व

नीलकमल = कर्मधारय (नीला जो कमल)
दशानन = बहुव्रीहि (दस मुँह वाला = रावण)
📝 Exam Q
"चतुर्भुज" किस समास का उदाहरण है?
(A) द्वंद्व (B) द्विगु (C) बहुव्रीहि (D) तत्पुरुष
✅ (C) बहुव्रीहि — चार भुजाओं वाला (विष्णु)

अलंकार (Alankar — Figures of Speech)
🔥 3–4 Q
›
📐 Key Alankar + Identifier
अनुप्रास = एक वर्ण बार-बार → "चारु चंद्र की चंचल"
उपमा = सरिस/जैसे/सा (explicit comparison)
रूपक = उपमेय = उपमान (no जैसे) → "मुख-चंद्र"
यमक = एक शब्द अलग-अलग अर्थ → "काली घटा"
श्लेष = एक शब्द, दो अर्थ एक साथ
अतिशयोक्ति = बढ़ा-चढ़ाकर कहना
⚡ Identify Trick
"जैसे/सा/सी/सरिस" दिखे → उपमा
No comparison word → रूपक
Same letter repeat → अनुप्रास
एक शब्द, 2 बार, 2 अर्थ → यमक
एक शब्द = 2 अर्थ → श्लेष

शब्दालंकार: अनु+यमक+श्लेष
अर्थालंकार: उपमा+रूपक+उत्प्रेक्षा

(A) उपमा (B) अनुप्रास (C) रूपक (D) श्लेष

Coding-Decoding (कूट लेखन)
🔥 4–6 Qहर साल
›
📐 4 Types of Coding
Type 1: Letter shift → COLD=DPME (+1 each letter)
Type 2: Number code → A=1,B=2...Z=26
Type 3: Reverse alphabet → A=Z,B=Y...Z=A
Type 4: Word/Symbol → अलग key दी होती है
Reverse word: APPLE → ELPPA
⚡ STEP-BY-STEP
Step 1: पहले word देखें, कितना shift है
Step 2: +1/-1 या ×2 pattern?
Step 3: Same shift सभी letters पर लगाएँ

Z के बाद A आता है (circular)
A=1 position: Z का reverse = A
Alphabet: EJOTY = 5,10,15,20,25
📝 Exam Q
यदि TEACHER = VGCEJGT, तो STUDENT = ?
(A) UVWFGPV (B) UVWFGOV (C) UVWFHPV (D) UVWEHPV
✅ (A) Each letter +2
यदि CAT=24, DOG=26, तो COT=?
(A) 35 (B) 36 (C) 37 (D) 38
✅ (B) C=3,O=15,T=20 → 3+15+20=38... check position sums

Direction Sense (दिशा ज्ञान)
🔥 3–4 Q
›
💡 Concept
एक व्यक्ति विभिन्न दिशाओं में चलता है। अंत में: वह किस दिशा में है? कितनी दूर है?

Compass: उत्तर (N), दक्षिण (S), पूर्व (E), पश्चिम (W)
Diagonal: NE, NW, SE, SW
⬆️ उत्तर
⬅️ पश्चिम    पूर्व ➡️
⬇️ दक्षिण
⚡ Tricks
Left/Right Turn:
उत्तर + दायाँ = पूर्व
उत्तर + बायाँ = पश्चिम
(घड़ी की दिशा = दायाँ)

Distance: Pythagoras लगाएँ
d = √(horizontal² + vertical²)

Shadow Rule:
🌅 सुबह shadow = पश्चिम में
🌇 शाम shadow = पूर्व में
📝 Exam Q
राम उत्तर 10km → पूर्व 6km → दक्षिण 10km चला। वह शुरू से कितना दूर, किस दिशा में?
(A) 4km पश्चिम (B) 6km पूर्व (C) 6km पश्चिम (D) 4km पूर्व
✅ (B) 6km पूर्व — उत्तर-दक्षिण cancel, केवल पूर्व 6km बचा
